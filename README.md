# mapdoge-bot

## Usage

Open [Google Maps](https://www.google.com/maps), select any palace and copy [Plus Code](https://maps.google.com/pluscodes/).

Send Plus Code to MapDogeBot, you will get MAPCODE.

**Only Japan Plus Code is valid.**
